scripts for scraping election results from ballotpedia for 2013 and 2014.

2013_2014_urls.csv contains the url for each page to scrape.

state_legislature_scrape_2013_2014.py is the scraper script.

election_results_2013_2014.csv is a copy of the output of the scraper file as run on Nov 3 2017, and can be used to generate the elections file if the scraper stops working.
